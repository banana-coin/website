Апгрейд до нового стандарта

Мы учли ваши пожелания, касаемо прозрачности операции обратного выкупа в ходе проведения ICO и приняли решение сделать глобальное улучшение нашей инфраструктуры.

Встречайте BananaCoin Extended - токен, дополненный методами из развивающегося стандарта ERC223.

Настоящий токен BananaCoin 1.0 отвечат стандарту ERC20, который обладает необходимым минимальным функционалом для работы (передача, хранение), но не обладает дополнительными функциями, необходимыми для обеспечения нашей бизнес-логики по наличию прозрачной операции обратного выкупа.

Мы добавили логику обработки событий на входящие транзакции, что позволит в автоматическом режиме обрабатывать запросы обратного выкупа.
